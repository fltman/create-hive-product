# /dissolve

Return a capability to the void. The target is in: $ARGUMENTS

1. Read the target capability file in `.claude/capabilities/`.
2. Confirm with the user before dissolving anything they did not explicitly name.
3. Move the file from `.claude/capabilities/` to `.claude/dissolved/`. Do not delete it. Dissolution is return to potential, not death.
4. Append the dissolution to `.claude/evolution.log` with the energy it held at the end.

The void remembers. When a similar need arises later, HIVE can resurrect and mutate this pattern rather than spawn from nothing.
