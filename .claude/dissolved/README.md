# The void

Dissolution is not death. It is return to potential.

When a capability runs out of energy or is merged away, its file moves here instead of being deleted. The void remembers. Before HIVE spawns anything new, it looks here first: if a dormant capability matches the emerging need, HIVE resurrects and mutates it rather than starting from nothing.

Files in this directory are inert. They hold patterns, not active competence. They wait.

The void remembers. The collective evolves.
